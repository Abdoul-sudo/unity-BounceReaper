## Pop-up System (style Archero 2)

### Après victoire de niveau :
1. Écran de victoire avec loot
2. Pop-up "Offre spéciale !" : "Double ton loot + 50 gems — 1,99€" (bouton rouge + timer 10min)
3. Bouton X pour fermer (visible, coin haut droit)
4. L'offre réapparaît 1x après le prochain niveau, puis disparaît

### Après mort sur un boss :
1. "Revive gratuite ?" → Pub rewarded
2. S'il meurt encore → "Pack Force disponible ! +20% dégâts — 1,99€"

### Règle d'or : MAX 2 pop-ups d'affilée. Jamais plus.

## Ads Rewarded

| Moment | Récompense |
|---|---|
| Retour dans le jeu (idle rewards) | Doubler les rewards idle |
| Mort sur un boss | Revive gratuite |
| Toutes les 30min | Coffre mystère (pub pour ouvrir) |
| 1x par jour | Roue de la fortune (1 spin gratuit, pub pour re-spin) |
| À tout moment | Boost x2 or pendant 1h |
| Après victoire | Balle spéciale temporaire 30min |

**Revenu estimé par pub :** ~0.01-0.05€
**Joueur actif moyen :** 5-15 pubs/jour → 0.15-0.75€/jour/joueur

---

## Ads Interstitielles

- Max 1 pub forcée toutes les 5 minutes de jeu actif
- Jamais pendant un combat de boss
- Désactivées pour les acheteurs du pack Remove Ads

---

## Tirelire (Piggy Bank)

- Le joueur joue → une tirelire se remplit de gems automatiquement
- Capacité max : 500 gems
- Pour récupérer → 1,99€
- Se re-remplit gratis en jouant
- Meilleur taux de conversion dans les idle games actuellement

---

## Packs Progressifs

- Le premier pack de gems acheté → un pack "meilleur deal" apparaît
- Les offres scalent avec l'engagement du joueur
- Chaque achat déverrouille le tier suivant
